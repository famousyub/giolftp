## 💡 要求
Go version: 1.9.x or later

## ⚡️ 快速开始
```bash
git clone https://github.com/voocel/ftf.git
cd ftf
go build .

./ftf send
./ftf receive
```

## ⚙️ 安装
从命令行安装
```
curl https://getftf.voocel.com | bash
```

## 🔥 功能
* [x] send
* [x] receive
* [x] progressbar
* [x] pretty terminal
